# ETL Mercados (Python)

Pipeline de dados simples (nível inicial) que lê arquivos **JSON** e **CSV** de duas “empresas”, **padroniza colunas**, **combina os dados** e salva um arquivo final em CSV na camada `data_processed`.

Este projeto foi feito para praticar fundamentos de Engenharia de Dados e boas práticas em Python, incluindo organização por pastas e orientação a objetos.

---

## Estrutura de Pastas

pipeline_dados/
├─ data_raw/ # Dados de entrada (brutos)
├─ data_processed/ # Saídas do pipeline (dados tratados)
├─ notebooks/ # Explorações e testes (Jupyter)
├─ scripts/ # Scripts do pipeline (execução)
└─ venv/ # Ambiente virtual (local)
